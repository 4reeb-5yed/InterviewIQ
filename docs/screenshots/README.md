# Screenshots

These are **placeholder SVGs** rendered in the root README. Replace them with real captures of the
deployed app (https://interview-iq-areeb-syed.vercel.app).

## How to add real screenshots

1. Capture these views (1280×800 recommended; PNG or WebP):
   - `landing` — the landing page (`/`)
   - `upload` — the upload screen (`/analyze`)
   - `report` — a completed report (`/analysis/:taskId`), light mode
   - `report-dark` — the same report in dark mode (toggle in the top bar)
2. Save them in this folder, e.g. `landing.png`, `upload.png`, `report.png`, `report-dark.png`.
3. In the root [`README.md`](../../README.md) Screenshots section, point the image links at the
   `.png` files (replace the `.svg` placeholders), then commit.

Tip: macOS `⌘⇧4`, Windows `Win+Shift+S`, or your browser devtools "capture full-size screenshot".
